
# 📚 StudyBuddy AI Agent

An AI-powered multi-agent student assistant designed to help students organize their academic life. StudyBuddy assists with study planning, reminders, motivation, and privacy protection through a collaborative multi-agent workflow.

---

## 🚀 Problem Statement

Students often struggle to manage study schedules, assignments, deadlines, and maintain motivation. StudyBuddy AI Agent simplifies student life by providing an intelligent assistant that organizes tasks while protecting sensitive information.

---

## ✨ Features

- 📅 Planner Agent – Creates and prioritizes study schedules.
- ⏰ Reminder Agent – Generates reminders for important tasks and deadlines.
- 💪 Motivation Agent – Provides encouraging messages to keep students motivated.
- 🔒 Security Agent – Protects sensitive information by masking personal details such as email addresses and phone numbers.

---

## 🤖 Multi-Agent Workflow

```
User
   ↓
Planner Agent
   ↓
Reminder Agent
   ↓
Motivation Agent
   ↓
Security Agent
   ↓
Final Response
```
